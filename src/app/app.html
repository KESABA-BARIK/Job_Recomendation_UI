About

Built for job seekers,
by a developer.

This is a full-stack ML project — a Python FastAPI backend running a trained classification model, paired with an Angular frontend. It extracts skills from uploaded resumes and matches them against a curated job dataset to surface the most relevant career paths.

The model was trained on a labelled dataset of tech resumes using TF-IDF feature extraction and a multi-class classifier with softmax confidence scoring.

Tech Stack
Angular TypeScript Python FastAPI scikit-learn TF-IDF Render
